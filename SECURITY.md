# Security Policy

## Supported Versions

Currently, only the latest version of the Distributed Agent Platform is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please do not report it via public issues. Instead, please follow the steps below:

1.  **Do not** open a public issue.
2.  Email your findings to [security@example.com](mailto:security@example.com) (Please replace this with your actual contact info).
3.  Include as much detail as possible, including steps to reproduce and potential impact.

We will acknowledge receipt of your report within 48 hours and provide a timeline for a fix if the vulnerability is confirmed.
